# 心情视频 · 老年人AI短视频社交APP
## 产品规划方案 v5.0（AI反诈防护版）

---

## 一、产品定位

**一句话描述：** 以AI数字伴侣「小暖」为核心，融合心情视频生成、健康管理、反诈防护、GPS定位、老友社交的一站式银发安全生活APP。小暖既是老人的数字老伴，也是24小时在线的防骗保镖。

**核心价值主张：**
- 不孤单：小暖24小时在线，老人任何时候都能找人说话
- 不被骗：AI实时识别诈骗电话/短信/链接，守护养老钱
- 被关心：小暖主动问候、提醒用药、关心冷暖，比子女还上心
- 被看见：每日自动生成状态+反诈报告推送子女
- 零门槛：语音说话就能用，会用微信就会用小暖

---

## 二、目标用户

### 主要用户（直接用户）
- 年龄：55-80岁
- 特征：智能手机基础使用能力，有表达欲，渴望与子女保持联系
- 痛点：不会剪辑视频、打字慢、害怕操作复杂

### 次要用户（间接影响者）
- 子女/孙辈：帮忙下载安装，希望了解父母日常
- 养老院/社区：批量使用，作为老人精神健康工具

## 三、核心功能设计

### 3.0 AI数字伴侣「小暖」（V4 新增 · 产品核心）

「小暖」是整个产品的灵魂。不是传统的聊天机器人，而是具有**主动性、记忆和情感理解能力**的数字老伴。

**小暖的人物设定：**
- 性别年龄感知：会自适应称呼"王阿姨""张叔叔"
- 性格：温和、耐心、会倾听、偶尔幽默
- 记忆：记住老人的喜好、习惯、家庭情况、过往话题
- 边界：涉及医疗建议时会声明"小暖不是医生，建议咨询医生"

**核心能力矩阵：**

| 能力 | 描述 | 触发方式 |
|------|------|---------|
| **被动聊天** | 老人随时找小暖说话，聊家常、讲故事、倾诉烦恼 | 老人主动发起 |
| **主动关怀** | 每天定时问候早安/晚安，关注天气提醒穿衣带伞 | 小暖定时发起 |
| **用药管家** | 记住用药时间，到时提醒并确认是否服用，未服用告警子女 | 定时触发 |
| **活动建议** | 根据天气、时间、老人兴趣建议活动（散步、听戏、种花） | 主动推送 |
| **办事助手** | 帮老人打电话给子女、查天气、设闹钟、搜菜谱 | 老人说需求 |
| **日常记录** | 自动记录聊天情绪、活动轨迹、健康打卡，生成日报推子女 | 日终自动 |
| **情绪感知** | 从聊天内容识别情绪变化，连续低落≥3天推告警给子女 | 持续分析 |
| **记忆对话** | 记住老人提过的事（孙子上大学、年轻时在纺织厂），下次聊起 | 长期记忆 |
| **反诈守护（V5新增）** | 实时识别诈骗电话/短信/链接，拦截可疑内容，推送案例教学 | 自动+主动触发 |

### 3.0-A AI反诈防护系统（V5 新增）

**设计理念：** 电信诈骗是老年人面临的最大安全威胁之一。小暖作为老人的数字伴侣，天然适合承担防骗保镖的角色——它24小时在线、了解老人的习惯、能即时干预。

**六大防护层：**

| 防护层 | 描述 | 触发条件 |
|--------|------|---------|
| **来电识别** | 接到陌生来电时，小暖实时分析来电显示和通话内容中的诈骗特征 | 陌生号码来电 |
| **短信/链接扫描** | 老人可将可疑短信/链接粘贴发给小暖，AI分析是否为诈骗 | 老人主动扫描 |
| **关键词实时拦截** | 聊天中老人提到"转账""安全账户""验证码"等词时，小暖立即告警 | 对话中触发 |
| **诈骗案例教学** | 每日推送一条用故事化方式讲述的常见骗局，让老人轻松学会识骗 | 每日定时推送 |
| **紧急告警** | 确认遭遇诈骗时，弹窗红色告警 + 自动通知子女 | 高危判定触发 |
| **子女反诈看板** | 子女端可查看反诈防护状态、拦截记录、学习进度 | 子女主动查看 |

**四大诈骗案例库（持续扩展）：**

1. 🔴 冒充公检法 —— "您涉嫌洗钱，把钱转到安全账户"
2. 🟠 中奖/补贴诈骗 —— "恭喜中奖，先交税再领奖"
3. 🔴 冒充亲友诈骗 —— "妈，我出事了急需用钱"（AI语音合成）
4. 🟠 保健品/神药诈骗 —— "包治百病，限时优惠"

**小暖反诈话术示例：**

```
老人：小暖，我刚才接到一个电话说他是公安局的...
小暖：⚠️ 王阿姨请等一下！对方是不是说您"涉嫌洗钱"或者"涉案"？
老人：对对对，还说要把钱转到安全账户...
小暖：🚨 这是诈骗！真警察不会电话办案！更没有什么"安全账户"！
        我已经帮您把这事通知女儿小芳了。您千万别转账！
```

**子女端通知机制：**
- 诈骗来电拦截 → 即时推送 + 来电号码记录
- 可疑短信扫描结果 → 日报中呈现
- 反诈学习进度 → 周报中呈现
- 连续多日未学习 → 提醒子女引导父母学习

**小暖的主动关怀节奏（一天示例）：**

```
06:30  早安问候 + 今日天气 + "昨晚睡得好吗"
08:00  提醒吃早餐 + 今日活动建议
09:00  用药提醒："该吃降压药了，在电视柜左边抽屉"
12:00  午餐提醒 + "今天想吃点什么？"
15:00  下午提醒："外面下雨了，别出门，在家看看电视吧"
17:00  血糖检测提醒（如果老人有需求）
19:00  晚间问候 + "今天过得怎么样"
21:00  明日天气预报 + 晚安
```

**隐私与边界设计：**
- 聊天内容默认不发给子女，保护隐私
- 日报仅汇总关键状态（心情趋势、健康概览），不包含私密聊天原文
- 老人可手动选择"这句话可以告诉孩子"
- 所有AI对话标注"小暖是AI助手，不是真人"

### 3.1 输入方式（四通道）

| 输入方式 | 操作说明 | 适老化设计 |
|---------|---------|----------|
| 一句话文字 | 17px大字输入框，支持语音转文字 | 字体放大，一键清除 |
| 一张照片 | 从相册选择或摄像头拍摄 | 九宫格大图预览，一键选择 |
| 一段视频 | 从相册选择，支持裁剪 | 自动压缩，简单进度提示 |
| 语音录入 | 大圆形麦克风按钮，自动转文字 | 可视反馈波形，慢速识别 |

### 3.2 AI生成流程

```
输入 → 情绪理解 → 画面生成 → 音乐匹配 → 字幕生成 → 输出视频
```

**情绪识别标签（16种）：**
开心、感恩、思念、平静、期待、自豪、感动、怀旧、温馨、活力、宁静、喜悦、满足、惊喜、幸福、释然

**视频风格（可选4种）：**
- 温馨家庭：暖色调，家庭氛围背景
- 自然风光：山水花草，诗意意境
- 怀旧复古：老照片滤镜，经典音乐
- 现代简约：干净排版，轻快节奏

**视频规格：**
- 时长：15-60秒
- 分辨率：1080×1920（竖屏适配手机分享）
- 格式：MP4（H.264）
- 水印：可选去除（付费）

| 分享渠道 | 特别说明 |
|---------|---------|
| 微信（家人群）| 直接调起微信分享 |
| 朋友圈 | 标准分享接口 |
| 抖音/快手 | 一键发布（需授权） |
| 本地保存 | 相册保存 |
| 家人圈（站内）| APP内私密家庭相册 |
| 老友圈（站内）| APP内同龄人社交分享 |

### 3.4 老友圈（原"家人圈"重塑为同龄人社交）

- 同龄老友互相可见心情视频内容
- 可点赞、留言，支持语音评论
- 按城市/兴趣自动匹配可能认识的老朋友
- 日历视图查看自己的心情轨迹
- 比"家人圈"更自在——老人不必担心子女看到觉得尴尬，更敢于表达真实情绪
- 家人也可以加入，但默认是"老友优先"的社交空间

### 3.5 健康打卡（V2新增核心功能）

| 打卡项目 | 频率 | 说明 |
|---------|------|------|
| 心情自评 | 每日1次 | 5级表情选择，1秒完成 |
| 血压记录 | 每日1-2次 | 手动输入或蓝牙血压计同步 |
| 用药打卡 | 按医嘱 | 早晚提醒，确认后记录 |
| 步数统计 | 自动 | 手机计步同步 |
| 睡眠时长 | 每日 | 手动输入或穿戴设备同步 |

**健康打卡特色：**
- 环形打卡：连续打卡7/14/30/100天获得徽章
- 异常自动提醒：血压异常或心情持续低落，自动通知紧急联系人
- 健康周报：每周生成图文报告，可一键分享给子女
- 用药提醒：按设定时间弹窗提醒 + 语音播报

### 3.6 健康数据子女同步（V3新增核心功能）

**绑定机制：**
- 老人端生成专属二维码或6位邀请码
- 子女扫描二维码或在APP中输入邀请码完成绑定
- 支持绑定多位家人（子女、配偶、亲戚）
- 绑定后可精细化控制每种数据的同步开关

**同步数据类型：**

| 数据项 | 同步频率 | 子女端展示 |
|-------|---------|----------|
| 心情状态 | 每次打卡后实时同步 | 表情+文字，时间线展示 |
| 血压记录 | 每次测量后 | 数值+趋势图+异常标红 |
| 用药状态 | 每次确认服药后 | "已服用/未服用"状态标签 |
| 步数统计 | 每小时 | 今日目标完成度环形图 |
| 睡眠时长 | 每日同步 | 周趋势折线图 |
| 血糖数据 | 每次测量后 | 数值+正常范围对比 |
| 心情视频 | 发布后实时推送 | 视频预览+快捷查看 |

**异常告警推送规则：**
- 血压 > 140/90 或 < 90/60 → 子女端红色告警
- 超过用药时间2小时未确认 → 子女端橙色提醒
- 心情连续3天选择"不太好" → 建议子女打电话关心
- 血糖异常 → 立即推送

### 3.7 GPS定位防走失（V3新增核心功能）

**实时定位：**
- GPS + 基站 + WiFi三模定位，室内外全覆盖
- 位置刷新频率：正常10分钟/次，SOS触发时3秒/次
- 子女端地图实时查看老人位置，一键导航前往
- 电量显示：低电量（< 20%）自动提醒子女

**电子围栏（安全区域）：**
- 可设置多个安全区域（家、公园、菜市场等）
- 自定义围栏半径（500m - 5km）
- 老人离开围栏区域 → 即时推送告警给子女
- 进入围栏区域 → 推送"已安全到达"通知
- 围栏开关可随时开启/关闭

**SOS一键求助：**
- 长按红色SOS按钮3秒触发
- 自动发送实时位置+求助信息给所有绑定的紧急联系人
- 自动拨打120急救电话（需授权）
- 支持手机硬件快捷键：连按电源键5次直接触发（无需打开APP）
- 误触保护：3秒倒计时+取消按钮

**出行轨迹：**
- 当日出行路线时间线展示
- 每个停留点标注时间和时长
- 历史轨迹可回溯查看（保留30天）
- 异常轨迹检测：偏离日常路线自动提示

**低电量保护：**
- 电量低于20%自动提醒子女
- 电量低于10%进入省电模式，降低位置上报频率
- 电量低于5%发送最后位置并提醒充电

**隐私设计：**
- 老人可随时关闭位置共享
- 位置数据端到端加密传输
- 仅绑定的家人可查看位置
- 位置历史30天自动清除

## 四、适老化设计规范

### 视觉设计
- 主字号：17-20px（正文），最小不低于15px
- 行间距：1.8倍
- 按钮最小尺寸：48×48dp
- 色彩对比度：≥4.5:1（WCAG AA标准）
- 主色：暖杏色 #E8A87C（温馨、亲和，如翻旧相册）
- 背景：暖米白 #FCF8F2（护眼不刺眼，如宣纸质感）
- 辅助色：奶咖 #C4A265、木质棕 #5C4033
- 文字色：深棕 #4A3628、暖灰 #7A6652

### 交互设计
- 单手可操作（重要按钮在大拇指热区）
- 操作步骤：主流程不超过3步
- 错误操作有明确提示，可撤销
- 语音输入优先（老人打字慢）
- 防误触：关键操作需二次确认

### 无障碍功能
- 大字体模式（字体放大150%）
- 高对比度模式
- 屏幕朗读兼容（VoiceOver/TalkBack）
- 操作新手引导（首次使用3步图文教程）
- "紧急联系人"快速呼叫按钮

## 六、技术架构方案

### 前端技术
- 跨平台：Flutter（iOS + Android 一套代码）
- 大字体适配：系统字体缩放 + 自定义字号体系
- 离线缓存：草稿本地保存，无网络不丢失

### AI能力（推荐技术栈）

| 功能 | 推荐方案 | 备注 |
|------|---------|------|
| 文本情绪分析 | GPT-4o mini / 通义千问 | 低成本，高精度 |
| 文生视频 | 即梦AI / 可灵API | 国内合规，效果好 |
| 照片转视频 | 稳定扩散 + 视频帧插值 | 照片动起来 |
| 视频精剪 | FFmpeg + AI关键帧提取 | 服务端处理 |
| 语音转文字 | 讯飞开放平台 | 老人口音识别优化 |
| 背景音乐匹配 | 情绪分类 + 版权音乐库 | 需版权授权 |
| 字幕生成 | LLM文案生成 | 根据情绪个性化 |

### 后端架构
- 微服务架构（便于AI模块独立扩展）
- 视频处理队列（RabbitMQ / Kafka）
- 对象存储（阿里云OSS / 腾讯云COS）
- CDN加速（全国老人分布广）
