# 心情视频 · 合作伙伴招募方案

> 目标：用已有的原型+规划文档，吸引对银发赛道感兴趣的合作伙伴加入开发

---

## 一、你的核心筹码（已有的）

别人的项目还在"我有一个想法"阶段，你已经有了：

| 资产 | 说服力 | 
|------|-------|
| 可交互的 V3 原型（双端+健康同步+GPS） | 让别人**3 秒看懂**这产品长什么样 |
| 完整产品规划 v3.0（定位/功能/适老化/商业模式） | 证明你**不是拍脑袋** |
| 工程实施计划（5 Phase、8 个月、成本模型） | 证明你知道**怎么做、花多少钱** |
| 233 个 AI 专家能完成 80% 技术工作 | 证明**开发成本极低** |

这些都是说服别人的硬通货。

## 二、找谁？四种最有价值的目标对象

### A 类：带钱的人（投资人 / 产业资本）

**他们关心什么：**
- 市场多大？（中国 60 岁以上 3 亿人，银发经济 7 万亿）
- 凭什么你行？（原型 + 规划 + AI 降本）
- 竞品是谁？（抖音/快手老年模式 ≠ 专为老人设计；小年糕是工具不是社交）

**去哪找：**

| 渠道 | 具体操作 |
|------|---------|
| **银发经济垂直 VC** | 投过"小年糕""糖豆广场舞"的机构：红杉、GGV、蓝驰、五源，找他们的银发赛道投资经理 |
| **腾讯/阿里产业基金** | 腾讯银发科技实验室、阿里达摩院智慧养老，他们对养老数字化有专项资金 |
| **天使投资人** | 在「小饭桌」「铅笔道」「创业邦」上发布项目 |
| **创业比赛** | 参加「银发经济创新创业大赛」或各地政府举办的养老科技路演 |

**给他们的 pitch 一句话：**

> "用 AI 把老人产生一条短视频的门槛降到一句话，3 亿银发人群的社交表达需求，目前没有一个产品真正解决。"

---

### B 类：带人的人（技术合伙人 / 创业伙伴）

**他们关心什么：**
- 技术挑战性？（AI 视频管线、GPS 定位、双端架构——有料）
- 自己投时间划不划算？（AI 专家承担 80% 开发，自己只需要把关+整合）
- 有没有前景？（银发赛道是未来 10 年确定性增长）

**去哪找：**

| 渠道 | 具体操作 |
|------|---------|
| **即刻 / V2EX** | 发帖："寻 Flutter 或 AI 方向的技术合伙人，银发经济赛道，已有完整原型和规划" |
| **GitHub / Gitee** | 把你的原型开源，写一篇吸引人的 README，技术人通过代码判断你的水平 |
| **大厂离职群** | 字节/腾讯/阿里内部有很多想做创业的人，找「离职创业互助群」 |
| **线下 Hackathon** | 参加「银发科技黑客松」或泛科技 Hackathon，现场组队 |

---

### C 类：带用户的人（养老机构 / 社区 / 医院）

**他们关心什么：**
- 能不能帮我的老人？（健康打卡、防走失、社交——每项都直击痛点）
- 要不要我开发？（不用，你做渠道伙伴就行）
- 怎么分钱？（合作分成）

**去哪找：**

| 渠道 | 具体操作 |
|------|---------|
| **养老机构** | 泰康之家、光大养老等连锁机构，找他们的数字化负责人 |
| **社区街道办** | 各地老龄办、街道养老服务中心，他们有钱买服务 |
| **老年大学** | 全国 7.6 万所老年大学，你的第一批种子用户的完美场景 |
| **保险公司** | 泰康、平安——健康数据+定位数据对他们的养老保险产品价值巨大 |

**给他们的 pitch 一句话：**

> "你们有老人，我们有让老人开心的工具。不需要开发，我们负责产品，你们负责推广，收入分成。"

## 三、去哪展示？具体操作路径

### 路径 1：开源社区冷启动（推荐作为第一步）

```
第 1 步：把原型+规划推到 GitHub/Gitee，写一篇高质量的 README
第 2 步：在 V2EX、即刻、掘金发帖："我用 AI 做了一个老年人心情视频 APP 原型，欢迎 Star 和参与"
第 3 步：星标过 500 后，自然有媒体和技术人关注
```

**为什么这条路好：**
- 零成本
- 技术人会通过代码质量判断项目价值
- 天然吸引同频的技术合作者
- 星标数据本身就能作为跟 VC 聊的筹码

### 路径 2：创业路演直通车

```
第 1 步：用原型录一段 2 分钟演示视频（手机录屏即可）
第 2 步：写一页纸的 pitch（用上面给 VC 的一句话模板）
第 3 步：找 3-5 个银发经济路演活动报名
第 4 步：路演现场，把原型打开让投资人自己点着玩
```

### 路径 3：直接接触潜在合作伙伴

```
养老机构：官网找「数字创新」或「战略合作」联系方式，发邮件附上原型链接
保险机构：找泰康在线/平安好医生的产品经理私信
老年大学：直接联系当地老年大学的校长或活动组织者
```

## 四、你还需要准备什么？

| 材料 | 用途 | 你现在的状态 |
|------|------|------------|
| 2 分钟路演视频 | 比赛/路演/邮件附上 | ❌ 还没有 |
| 一页纸 BP（商业计划书） | 给投资人看 | ❌ 还没有 |
| 官网/Landing Page | 让别人自己了解 | ❌ 还没有 |
| 竞品分析 | 证明差异化 | ✅ 产品规划里有 |
| 财务模型 | 证明能赚钱 | ✅ 成本分析做了 |

**建议优先级**：先做一个简单的 Landing Page + 2 分钟演示视频，这两个东西有了之后，你发任何邮件都能让人 30 秒内看懂这产品。

---

## 五、一个可行的节奏建议

```
本周：录一个 2 分钟原型演示视频
下周：做一个简单的 GitHub README + Landing Page
第三周：在即刻/V2EX/掘金发帖开源
第一个月：同时找 2-3 个养老机构做试点沟通
第二个月：报名 1-2 个创业路演
第三个月：根据反馈决定是否需要全职投入
```

## 六、最关键的一句话

> **你不需要先有一个公司，才能找合作伙伴。你需要的是一份让人 30 秒就能看懂的演示，和 3 个愿意试用的老年用户。**

原型你已经有了，接下来就是把它们推到正确的渠道上。
